AstroReha REST API Quick Reference

AstroReha is designed to be high-performance, modular, and extremely accurate, leveraging Swiss Ephemeris under-the-hood. The following endpoints are exposed for third-party integrations and development:

POST /api/birth-chart

Generates the full Birth Chart planetary coordinates and Lagna information.

POST /api/getBirthNavamsaChart

Generates both D1 Birth and D9 Navamsa chart positions in a single request.

POST /api/navamsa-chart

Recalculates Navamsa (D9) positions given raw birth chart positions.

POST /api/compatibility

Runs the standard 30-point synastry evaluation between two distinct charts.

GET /api/planets

Retrieve raw astronomical planetary positions using Swiss Ephemeris.
